# Ayurvedic Prakriti Classification

## Overview
This project predicts the Ayurvedic Prakriti type using Machine Learning techniques based on wellness and psychological data.

## Features
- Data preprocessing
- Machine Learning model training
- Prakriti classification
- Tkinter-based GUI

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Scikit-learn
- Tkinter

## Files
- finalcode.ipynb – Main machine learning notebook
- tkinter1.ipynb – GUI implementation
- ayurveda_psychology_wellness_dataset.csv – Dataset
- ayurvedic Image.jpeg – Sample output image
